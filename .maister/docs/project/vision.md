# Project Vision

## Overview
Maciej Harbuz Resume is a professional online CV that showcases experience, skills, projects, and certifications through a polished single-page website.

## Current State
- **Age**: Multi-year project (30+ commits, active since initial creation)
- **Status**: Active development with regular content and feature updates
- **Users**: Recruiters, hiring managers, and professional contacts
- **Tech Stack**: Pug, SCSS, Bootstrap 5, Node.js build system, GitHub Pages

## Purpose
This project exists to provide a professional, always-accessible online resume that presents Maciej Harbuz's professional experience, technical skills, education, and certifications in an engaging, responsive format. It replaces static PDF resumes with a living, maintainable web presence.

## Goals (Next 6-12 Months)
- Keep resume content current with latest experience and skills
- Improve visual design and add new interactive features
- Maintain responsive, accessible design across all devices

## Evolution
Started as a customization of the Start Bootstrap Resume template, the project has evolved with a custom modular build system, automated CI/CD deployment, and personalized content sections including projects, certificates, and interests. The focus continues to be on content freshness and design improvements.
